# Constituency Pulse Page

A weekly, owned-content summary: "Top 5 issues in [constituency] this week."
Only draft for the 5–10 most active constituencies (query the issues table for
the past 7 days, rank by summed engagement). Load state-brand-voice alongside.

## Format

**Title:** Top 5 issues in [Constituency] this week — [date range]

**Intro** (2 sentences): neutral framing — what residents have been talking about,
based on public local discussion. Name the MP once, factually.

**Per issue (×5):**
- **[Issue headline, ≤ 10 words]**
- 2–3 sentences: what residents are raising (aggregate, no usernames/quotes),
  where exactly, who's responsible (council dept / MP), and any known process
  status (consultation open, works scheduled) — only if verifiable.
- If a related motion exists on state.com: "There's a live motion asking
  [MP/councillor] to [ask] → [link]". If not, note the gap for Mitesh — a Pulse
  issue without a motion is a motion-seeding candidate.

**Footer (mandatory):** how it's made — "Compiled by state.com from public local
discussion across [sources], aggregated and anonymised. We never track
individuals." Plus a link to start a motion.

500–800 words total. Specific, factual, neutral. Thin-content check: if you can't
write 2 substantive sentences about an issue, it doesn't go on the page.

## Process
1. Query the week's issues for the constituency. 2. Draft. 3. Hand to Mitesh for
edit + publication on state.com. You never publish.
